# TGD Soundboard - Feature Roadmap

Track the development progress of new features.

## Completed

- [x] Sound clip playback with categories
- [x] Virtual audio cable integration (VB-Cable)
- [x] Audio mixer with level meters
- [x] Monitor/preview mode (hear virtual cable output)
- [x] Per-app audio routing
- [x] Clip editor with waveform trimming
- [x] Multiple themes (Purple, Cyan, Neon)
- [x] Auto-updates
- [x] Rename clips
- [x] Custom window chrome

---

## Quick Wins (v1.5.0)

- [x] **Clip Color Picker** - Customize individual clip button colors (right-click > Color)
- [x] **Double-click to Rename** - Quick access to rename dialog
- [x] **Duplicate Clip** - Copy an existing clip with different settings (right-click > Duplicate)

---

## High Priority (v1.6.0)

- [x] **Global Hotkeys** - Trigger clips with keyboard shortcuts even when app isn't focused (right-click > Set Hotkey)
- [x] **Search/Filter** - Quickly find clips by typing in the search box
- [x] **Drag & Drop Reorder** - Rearrange clips by dragging them
- [x] **Per-Clip Volume** - Set different default volumes for each clip (right-click > Volume)

---

## Medium Priority (v1.7.0)

- [x] **Loop Mode** - Toggle to loop a clip continuously (right-click > Loop)
- [x] **Fade In/Out** - Smooth fade effects for clip playback (right-click > Fade Effects)
- [x] **Favorites Bar** - Pin most-used clips to a quick access bar (right-click > Favorite)
- [x] **Random Play** - Play a random clip from a category (shuffle button)

## Advanced Features (v1.8.0)

- [ ] **OBS WebSocket Integration** - Trigger OBS scenes/sources when playing clips
